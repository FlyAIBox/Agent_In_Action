## 💡 输入技巧

### ✅ 推荐格式

包含以下信息效果最佳：
1. **目的地**: 明确的城市名称
2. **时间**: 具体日期或天数
3. **人数**: 几个人
4. **预算**: 大概预算或预算类型
5. **兴趣**: 喜欢什么活动

### ⭐ 完美示例
```
我和老婆想8月去杭州玩5天，预算每人5000元左右，
喜欢自然风光、美食和文化体验，住精品酒店
```

### ❌ 避免这样说
```
"想去玩玩"  →  太模糊
"随便"      →  无法理解
"不贵就行"  →  预算不明确
```

## 📱 典型使用流程

### 场景1：快速规划（30秒）

1. 打开页面
2. 输入："北京3日游，历史文化"
3. 旅小智自动创建任务
4. 等待3-5分钟
5. 获取完整规划报告

### 场景2：详细定制（2分钟）

1. 点击侧边栏"开始规划"

3. 选择兴趣爱好、预算等
4. 点击"开始规划"
5. 获取个性化方案

### 场景3：多轮对话（5分钟）

1. 输入："我想去旅游"
2. 旅小智问："请告诉我目的地"
3. 回答："杭州"
4. 旅小智问："什么时候？几天？"
5. 回答："下周，3天"
6. ...继续补充信息
7. 完成规划

## 🔧 API端点

如果你是开发者，可以直接调用API：

### 自然语言接口
```bash
curl -X POST http://localhost:8080/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "我想去北京玩3天"}'
```

### 创建任务
```bash
curl -X POST http://localhost:8080/plan \
  -H "Content-Type: application/json" \
  -d '{ /* 旅行数据 */ }'
```

### 查询状态
```bash
curl http://localhost:8080/status/{task_id}
```

### API文档
```
http://localhost:8080/docs
```

## ❓ 常见问题

### Q1: 旅小智没有理解我的需求怎么办？
**A**: 请尝试：
1. 提供更具体的信息（目的地、时间）
2. 使用完整的句子
3. 或者使用左侧表单详细填写

### Q2: 可以修改旅小智的回复吗？
**A**: 目前一轮对话完成后，建议重新输入或使用表单调整。

### Q3: 支持哪些语言？
**A**: 目前仅支持中文，英文支持即将上线。

### Q4: 规划需要多长时间？
**A**: 通常3-5分钟，取决于目的地复杂度。

### Q5: 如何下载规划报告？
**A**: 规划完成后，页面会显示下载按钮，支持Markdown和JSON格式。
