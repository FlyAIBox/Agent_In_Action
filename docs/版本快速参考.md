## 🔄 版本升级指南

### 安全升级 (推荐)
```bash
# 仅升级补丁版本，保持兼容性
pip install --upgrade langchain==0.3.*
```

### 完整升级 (谨慎)
```bash
# 升级到最新版本 (可能引入 Breaking Changes)
pip install --upgrade langchain langgraph
```

### 版本降级
```bash
# 回退到指定版本
pip install langchain==0.3.27 --force-reinstall
```

## ⚠️ 常见问题

### Q: 版本冲突怎么办?
**A**: 使用独立虚拟环境
```bash
conda create -n project1 python=3.10.18 -y
conda activate project1
```

### Q: 安装速度慢?
**A**: 使用国内镜像源
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple [package]
```

### Q: 如何查看当前环境所有版本?
**A**: 导出依赖清单
```bash
pip freeze > current_versions.txt
```
